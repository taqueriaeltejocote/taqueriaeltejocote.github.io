# EL TEJOCOTE

Menú digital de la taquería **El Tejocote**.

## Descripción

Página web ligera diseñada para mostrarse en teléfonos móviles mediante un código QR.

Incluye:

- Menú con imágenes
- Animaciones al hacer scroll
- Botón de pedidos por WhatsApp
- Ubicación mediante Google Maps

## Uso

Escanea el código QR para abrir el menú desde tu celular.

## Tecnologías

- HTML
- CSS
- JavaScript
